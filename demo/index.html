Categorical or Nominal represent types of data which may be divided into groups.
Examples of categorical variables are color, sex, age group, and educational level.

Please enter categorical values for your data, separated by a semi-colon:
For example, for Color, it is: red; green; blue

Ordinal data is a categorical, statistical data type where the variables have natural, ordered categories and the distances between the categories is not known.

Please enter ordinal values for your data, separated by a semi-colon:
For example, for Emotions, it is: Very unhappy; Unhappy; OK; Happy; Very happy

Interval data, also called an integer, is defined as a data type which is measured along a scale, in which each point is placed at equal distance from one another. Interval data always appears in the form of numbers or numerical values where the distance between the two points is standardized and equal.
